# 一、C语言简介

## 1.1 地位

# 三、第一个C语言程序

gcc：UNIX家族操作系统通用C语言编译器 ――――Intel系列CPU

gcc 源码文件(.c)  -o 可执行文件名 -Wall

Linux下习惯上可执行文件名不加任何扩展名



```c
#include <stdio.h> //在本程序中，是为了给printf函数调用加函数声明

//main函数定义
int main()   //函数头     
{//函数体开始
	printf("Hello World\n"); //printf来自C库，其功能是将“”中内容显示在命令行界面上
	return 0;//main函数中返回0表示程序正常运行结束，其它值表示程序没能正常运行结束
}//函数体结束

```

int 整型数  用作函数返回值类型时，表示该函数返回一个整型数字，这样函数体内必须得以return语句作为结束，return语句的写法`return 整型表达式;` 

void 空类型  用作函数返回值类型时，表示该函数没有返回值，这样函数体内如果用到return语句，return语句的写法`return;`,也可以没有return语句，此时函数体最后一条语句执行结束就表示函数自动返回

一个函数定义的组成：
1). 函数头
   函数返回值类型    函数名()      --------- 无参函数
2). 函数体
   {
	//多条语句
   }

main函数作为整个程序的入口函数，任何一个C语言程序有且仅有一个叫做main的函数

# 四、第二个C语言程序

```c
#include <stdio.h>

void PrintHello();//函数声明，C语言语法规定：调用一个函数前，要先声明一下该函数，其作用是协助编译器检查函数调用写得是否正确
void PrintWorld();//因此函数声明语句只在编译过程其作用，与程序的运行无关
int main()
{
	PrintHello();//所谓的函数调用就是指使用指定函数的功能，调用一次就是使用一次函数的功能
	PrintWorld();
	return 0;
}

void PrintHello()
{
	printf("Hello ");
}

void PrintWorld()
{
	printf("World\n");
}
```



学习函数需要关注函数的三个方面：

1. 函数定义
2. 函数调用
3. 函数声明

# 八、语句分类

C语言程序主体构成是函数，但内容上是由函数体内和函数体外的语句构成



## 8.1 执行语句

## 会被编译器翻译成二进制代码，在程序执行过程中完成相应的动作

比如：函数调用语句
          函数定义

a.  单条的独立语句          以分号结尾
b.  复合语句              以}结尾不用加分号结尾，不是以}结尾则需加分号结尾
    

b.1 无头无尾的复合语句  --------- 普通复合语句

```
{
	//其它语句
}
```

b.2 带头部的复合语句

```
头部
{
	//其它语句
}
```

b.3 带头带尾的复合语句

```c
do
{
	//其它语句
}while();
```

## 8.2 非执行语句：

## 不会被编译器翻译成二进制代码，只是在生成可执行文件的过程(编译)中起作用，程序执行过程中不起作用

## 	比如：#inlude语句  函数声明语句

```
#开头的非执行语句，不以分号结尾
不以#开头的非执行语句，均以分号结尾
```



# 九、注释

语法上讲是可有可无的，跟程序逻辑没有关系，在编译过程中被编译器直接忽略
其作用是为了增强代码的可阅读性

行尾注释：(也可以单独一行)
     //..... 

多行注释：(不能嵌套)
/*
....
......
*/
	

# 十、C语言代码的编译全过程

?								预处理                    编译                      汇编                                     链接

f1.c                          f1.i                         f1.S                        f1.o

f2.c                          f2.i                          f2.S                      f2.o 

.....

fn.c                          fn.i                         fn.S                       fn.o     





预处理：处理.c文件中#开头的语句

编译：将C语言语句翻译成同功能的汇编指令

汇编：将汇编指令翻译成CPU的二进制指令

链接:  按特定规则将所有目标文件打包成与操作系统相匹配的可执行文件
